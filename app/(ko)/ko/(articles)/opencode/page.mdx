# OpenCode란 무엇인가?

OpenCode는 Anthropic의 Claude Code와 유사하게 터미널에서 직접 실행되는 오픈 소스 AI 코딩 에이전트입니다. 개발자의 워크플로우를 이해하고 코드 작성, 리팩토링, 버그 수정 등을 지원합니다.

## OpenCode의 주요 특징

- **터미널 중심**: IDE를 떠날 필요 없이 명령줄에서 바로 AI와 협업할 수 있습니다.
- **오픈 소스**: 누구나 코드를 확인하고 기여할 수 있는 투명한 개발 모델을 따릅니다.
- **강력한 기능**: 파일 읽기, 쓰기, 셸 명령 실행 등 프로젝트 전체를 제어할 수 있는 권한을 가집니다.
- **비용 효율적**: 사용한 만큼 지불하거나 로컬 모델을 연동하여 사용할 수 있습니다.

---

## 설치 방법 (How to install)

OpenCode를 설치하려면 시스템에 **Node.js 18** 이상이 설치되어 있어야 합니다. 터미널에서 다음 명령어를 실행하세요:

```bash
npm install -g opencode
```

설치가 완료되면 터미널에 `opencode`를 입력하여 시작할 수 있습니다:

```bash
opencode
```

처음 실행 시 Anthropic API 키 또는 로컬 모델 설정을 안내합니다.
