# OpenCode MCP Tools & Integrations

This page lists **popular MCP tools and integrations** you can use with OpenCode.

If you are new to MCP, start with:
→ [What is MCP in OpenCode](/en/mcp)

---

## Browser Automation

### Playwright MCP
Automate browsers for testing, scraping, and interaction.

Use cases:
- End-to-end testing
- Web scraping
- UI interaction

→ [Playwright MCP Guide](/en/opencode-mcp-playwright)

---

## Code & Repository Access

### GitHub MCP
Allow models to read repositories, issues, and pull requests.

Use cases:
- Code review
- Repository analysis
- Refactoring workflows

---

## Design Tools

### Figma MCP
Let models inspect design files and components.

Use cases:
- Design review
- UI consistency checks
- Spec validation

---

## Workflow Automation

### n8n MCP
Trigger workflows and automation pipelines from OpenCode.

Use cases:
- Task orchestration
- Internal automation
- Integration with third-party services

---

## Custom MCP Servers

You can build **custom MCP servers** to expose internal tools or systems.

Typical scenarios:
- Internal APIs
- Company dashboards
- Proprietary workflows

---

## How to Choose the Right MCP Tool

Use MCP tools when your task requires:

- Interaction with external systems
- Multi-step automation
- Persistent state or workflows

For protocol-level details, see:
→ [OpenCode MCP Overview](/en/mcp)
