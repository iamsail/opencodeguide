# 命令 (Commands)

## 这是什么？
命令 (Commands) 配置允许你定义自定义的 "Slash Commands"（斜杠命令）。就像在聊天框输入 `/explain` 或 `/test` 一样，你可以创建自己的 `/my-script` 来触发预设的 Prompt 或脚本。

## 什么时候你会用到它？
- 当你发现自己经常重复输入同一段复杂的 Prompt（比如：“请帮我按以下格式写单元测试...”）时，可以将其封装为 `/test-gen`。
- 当你需要一键触发某个项目特定的构建脚本并让 AI 分析输出时。
- 当你想为团队定义一套标准的操作指令集时。

## 核心概念
- **Slash Command**：以 `/` 开头的触发词。
- **Alias**：命令的简写别名。
- **Template**：命令背后对应的预设文本模版或动作序列。

## 常见误解 / 注意事项
- 自定义命令的名字不要与系统内置命令（如 `/help`, `/clear`）冲突。
- 命令不仅可以填入文本，还可以配置为自动附带特定的上下文文件。
