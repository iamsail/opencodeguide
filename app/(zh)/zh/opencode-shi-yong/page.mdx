# OpenCode 使用指南：新手 10 分钟上手

这是一份 **面向新手的 OpenCode 使用教程**。  
不讲概念，直接带你从 0 到第一次成功使用。

---

## 使用 OpenCode 前需要准备什么？

在开始之前，你需要：

- 一台本地电脑（macOS / Linux 更友好）
- 基本的命令行使用经验
- 已安装常见开发环境（如 Node / Python）

---

## 第一步：安装 OpenCode

通常通过命令行安装（以官方推荐方式为准）：

```bash
# 示例（具体命令请以官方文档为准）
npm install -g opencode
```

安装完成后，确认是否成功：

```bash
opencode --version
```

如果能看到版本号，说明安装成功。

---

## 第二步：配置模型

OpenCode 的一大优势是 **模型可配置**。

你可以选择：

* 本地模型（如 Ollama）
* 远程 API（如 OpenAI、其他兼容接口）

常见配置步骤包括：

* 设置模型名称
* 配置 API Key
* 指定运行方式

完成后，OpenCode 就知道该“用哪个大脑”了。

---

## 第三步：开始使用 OpenCode

进入你的代码目录：

```bash
cd your-project
opencode
```

你现在可以直接输入指令，例如：

* “解释这个项目的结构”
* “帮我重构这个函数”
* “这个报错是什么意思？”

OpenCode 会结合当前代码上下文给出回答。

---

## 一个真实使用示例

场景：你接手一个陌生项目。

你可以直接问：

> “这个项目的主要模块是什么？”

OpenCode 会扫描代码结构并给出总结，比你自己翻文件要快得多。

---

## 常见问题

### OpenCode 装不上怎么办？

* 检查 Node / 环境版本
* 确认是否有权限问题
* 查看官方安装说明

### 模型没反应？

* 检查 API Key
* 检查模型名称是否正确
* 确认网络或本地模型是否运行
