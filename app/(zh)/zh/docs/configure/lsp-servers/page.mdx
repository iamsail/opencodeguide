### 配置 (Configure)

您可以在 `opencode.json` 的 `lsp` 部分进行自定义：

```json
{
  "lsp": {
    "typescript": {
      "disabled": false,
      "command": ["typescript-language-server", "--stdio"],
      "extensions": [".ts", ".tsx"]
    }
  }
}
```

### 禁用 LSP 服务器 (Disabling)

* **全局禁用：** 将 `lsp` 设置为 `false`。
* **禁用特定服务器：** 在对应服务器配置中设置 `"disabled": true`。

### 自定义 LSP 服务器 (Custom)

您可以手动添加新的 LSP 服务器：

```json
{
  "lsp": {
    "my-custom-lsp": {
      "command": ["custom-lsp-server", "--stdio"],
      "extensions": [".custom"]
    }
  }
}
```
