# 代理技能 (Agent Skills)

**通过 SKILL.md 定义可复用的行为**

代理技能允许 OpenCode 从您的仓库或主目录中发现可复用的指令。技能通过原生的 `skill` 工具按需加载——代理能看到可用技能，并在需要时加载完整内容。

### 文件放置位置

每个技能对应一个文件夹，内部放置一个 `SKILL.md`。OpenCode 搜索路径如下：

* **项目级：** `.opencode/skill/<name>/SKILL.md`
* **全局级：** `~/.config/opencode/skill/<name>/SKILL.md`
* **Claude 兼容路径：** `.claude/skills/<name>/SKILL.md`

### 编写 Frontmatter

每个 `SKILL.md` 必须以 YAML frontmatter 开头。必需字段包括：

* `name`: 1–64个字符，小写字母、数字及单连字符。需与目录名一致
* `description`: 1–1024个字符，描述需足够具体以便代理选择

### 权限与禁用

您可以在 `opencode.json` 中控制代理对技能的访问：

```json
{
  "permission": {
    "skill": {
      "*": "allow",
      "internal-*": "deny",
      "experimental-*": "ask"
    }
  }
}
```

* `allow`: 立即加载
* `deny`: 对代理隐藏
* `ask`: 加载前提示用户
