# 生态系统 (Ecosystem)

## 这是什么？
OpenCode 生态系统不仅包含官方的核心组件，还包括社区贡献的各类资源。这里汇集了第三方客户端、MCP 服务器实现、Prompt 模版库以及各种语言的适配层。

## 什么时候你会用到它？
- 当你在寻找是否有现成的 MCP Server 可以连接到你的 Notion 或 Jira 时。
- 当你想看看有没有针对 Rust 或 Elixir 语言优化的 System Prompt 配置时。
- 当你希望能参与到 OpenCode 开源社区的贡献中来时。

## 核心概念
- **Registry**：集中索引社区插件和工具的市场。
- **Awesome OpenCode**：精选的资源列表。
- **Community Standards**：保证组件互操作性的规范文档。
