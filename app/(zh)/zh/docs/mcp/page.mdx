# MCP 协议概览

MCP（Model Context Protocol）是 OpenCode 的核心协议，用于定义模型、智能体与工具之间如何共享和更新上下文。

## MCP 在 OpenCode 中的作用

MCP 负责：

- 上下文结构化与持久化
- 工具调用的边界与权限控制
- 多步任务的执行一致性
