# 分享 (Share)

## 这是什么？
分享功能允许用户将与 OpenCode 的对话记录、生成的代码片段或解决问题的过程导出并分享给他人。支持生成 Markdown 文件、图片或在线链接，方便团队协作和知识沉淀。

## 什么时候你会用到它？
- 当你通过 AI 解决了一个棘手的 Bug，希望将解决思路和过程分享给团队成员时。
- 当你需要将 AI 生成的代码方案提交给同事进行 Code Review 时。
- 当你在技术博客或社区撰写教程，需要引用 AI 的回答时。

## 核心概念
- **会话导出**：将完整的问答上下文导出为 Markdown 或 JSON 格式。
- **隐私脱敏**：在分享前自动识别并隐藏可能的敏感信息（如 API Key, 密码）。
- **永久链接**：生成一个可访问的网页链接，展示格式化后的对话内容。

## 常见误解 / 注意事项
- 默认情况下分享是手动的，OpenCode 不会自动公开你的任何对话记录。
- 分享不仅包含结果，建议也包含 Prompt 过程，这对于复现结果非常有帮助。
