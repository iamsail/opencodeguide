# OpenCode 会话共享 (Share)

OpenCode 的共享功能允许您为会话创建公开链接，以便与团队成员协作或寻求帮助。

### 工作原理

当您共享会话时，OpenCode 会：

1. 为您的会话创建一个唯一的公开 URL。
2. 将您的对话历史同步到服务器。
3. 通过 `opncd.ai/s/<share-id>` 链接使对话可供访问。

### 共享模式

OpenCode 支持三种共享模式，通过 `opencode.json` 配置文件进行管理：

* **手动模式 (默认)**
会话不会自动共享。您需要手动输入命令生成链接：
```bash
/share
```
链接会自动复制到剪贴板。

* **自动共享 (Auto-share)**
在配置文件中设置 `"share": "auto"`。启用后，每个新会话都会自动生成共享链接。

* **禁用 (Disabled)**
在配置文件中设置 `"share": "disabled"` 可完全禁用共享功能。

### 取消共享

输入以下命令即可删除共享链接并销毁服务器上的相关数据：

```bash
/unshare
```

### 隐私建议

* 仅共享不含敏感信息的对话。
* 共享前检查对话内容。
* 避免共享包含专有代码或机密数据的会话。
* 协作完成后及时取消共享。
- 分享不仅包含结果，建议也包含 Prompt 过程，这对于复现结果非常有帮助。
