# 使用概览 (Usage)

## 这是什么？
使用概览提供了 OpenCode 不同交互模式的整体介绍，帮助用户理解如何高效地与 AI 助手协作。无论你是习惯使用命令行、IDE 插件还是独立的桌面界面，这里都能找到适合你的工作流。

## 什么时候你会用到它？
- 当你是初次使用者，想要了解 OpenCode 提供了哪些交互界面时。
- 当你想探索如何将 OpenCode 更好地融入你现有的开发环境（如 VS Code 或 Neovim）时。
- 当你需要了解基本的指令格式和交互逻辑时。

## 核心概念
- **交互模式 (Modes)**：OpenCode 支持终端 UI (TUI)、命令行 (CLI) 和 IDE 插件等多种运行方式。
- **指令 (Prompts)**：用户输入的自然语言命令，用于驱动 AI 执行任务。
- **上下文 (Context)**：用户当前选中的文件、代码块或整个项目状态，是 AI 理解任务的基础。

## 常见误解 / 注意事项
- OpenCode 并不仅仅是一个聊天机器人，它具备读取、编辑文件和执行系统命令的能力。
- 不同模式下的核心功能是一致的，但交互体验可能会有差异。
