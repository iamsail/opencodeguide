# IDE

OpenCode 是适用于 VS Code、Cursor 以及其他 IDE 的扩展。

## 使用方法

- **快速启动**：使用 `Cmd+Esc`（Mac）或 `Ctrl+Esc`（Windows/Linux）在分屏终端视图中打开 OpenCode；如果已经有运行中的会话，则会聚焦到现有终端会话。
- **新建会话**：使用 `Cmd+Shift+Esc`（Mac）或 `Ctrl+Shift+Esc`（Windows/Linux）启动一个全新的 OpenCode 终端会话（即使已经有一个在运行）。你也可以点击界面中的 OpenCode 按钮。
- **上下文感知**：会自动把当前选中的代码或当前打开的标签页内容分享给 OpenCode。
- **文件引用快捷键**：使用 `Cmd+Option+K`（Mac）或 `Alt+Ctrl+K`（Linux/Windows）插入文件引用。例如：`@File#L37-42`。

## 安装方法

要在 VS Code 及其常见分支版本（Cursor、Windsurf、VSCodium 等）上安装 OpenCode：

1. 打开 VS Code（或对应 IDE）
2. 打开集成终端
3. 直接运行命令：
   ```bash
   opencode
   ```
   → 扩展会自动安装完成

**另一种场景**：如果你希望在 TUI 中使用 `/editor` 或 `/export` 命令时打开你自己的 IDE，需要设置环境变量：

```bash
export EDITOR="code --wait"
```

（更多关于 TUI 中编辑器设置的内容可参考：[/zh/docs/tui/#editor-setup](/zh/docs/tui/#editor-setup)）

### 手动安装

在 VS Code 的扩展市场（Extensions Marketplace）中搜索 **OpenCode**，然后点击 **Install** 安装即可。

## 故障排除

如果扩展未能自动安装，请检查以下几点：

- 确保你是在**集成终端**中运行的 `opencode` 命令
- 确认你的 IDE 已正确安装对应的 CLI 命令：
  - VS Code → `code` 命令
  - Cursor → `cursor` 命令
  - Windsurf → `windsurf` 命令
  - VSCodium → `codium` 命令
- 如果缺少对应命令：
  - 按 `Cmd+Shift+P`（Mac）或 `Ctrl+Shift+P`（Windows/Linux）打开命令面板
  - 搜索并执行 “Shell Command: Install ‘code’ command in PATH”（或你所用 IDE 的对应命令）
- 确保 VS Code（或对应 IDE）有权限安装扩展
