# 介绍 (Intro)

## 这是什么？
OpenCode 是一个开源的智能编程助手，旨在通过自然语言交互来辅助开发者完成代码编写、调试和重构等任务。它不仅支持主流的商业大模型（如 GPT-4、Claude 3.5 Sonnet），还完美兼容本地模型（如 Ollama），为开发者提供灵活、高效且隐私安全的编程体验。

## 什么时候你会用到它？
- 当你需要快速理解复杂的代码逻辑或项目结构时。
- 当你希望通过对话方式自动生成代码片段或完整功能模块时。
- 当你需要对现有的代码进行重构、优化或查找潜在 Bug 时。

## 核心概念
- **自然语言交互**：直接使用中文或英文描述你的需求，OpenCode 会自动转换为代码操作。
- **多模型支持**：无缝切换云端商业模型和本地开源模型。
- **上下文感知**：能够理解当前项目的代码结构和上下文信息，提供更精准的建议。

## 常见误解 / 注意事项
- OpenCode 不是 IDE，而是一个可以集成到 IDE 或终端中的辅助工具。
