# OpenCode 文档（中文版）

**OpenCode** 是一个开源的 AI 编码代理，支持终端界面、桌面应用或 IDE 扩展使用。

## 前提条件

要在终端中使用 OpenCode，需要：
1. 一个现代终端模拟器，例如：
   - [WezTerm](https://wezterm.org)（跨平台）
   - [Alacritty](https://alacritty.org)（跨平台）
   - [Ghostty](https://ghostty.org)（Linux 和 macOS）
   - [Kitty](https://sw.kovidgoyal.net/kitty/)（Linux 和 macOS）
2. 所用 LLM 提供商的 API 密钥。

## 安装

安装 OpenCode 最简单的方法是通过安装脚本：

```bash
curl -fsSL https://opencode.ai/install | bash
```

其他安装方式：

**使用 Node.js**

```bash
npm install -g opencode-ai
```

**macOS 和 Linux 使用 Homebrew**

```bash
brew install anomalyco/tap/opencode
```

**Arch Linux 使用 Paru**

```bash
paru -S opencode-bin
```

**Windows**

- **Chocolatey**：`choco install opencode`
- **Scoop**：`scoop bucket add extras` 然后 `scoop install extras/opencode`
- **NPM**：`npm install -g opencode-ai`
- **Mise**：`mise use -g github:anomalyco/opencode`
- **Docker**：`docker run -it --rm ghcr.io/anomalyco/opencode`

也可以从 [Releases](https://github.com/anomalyco/opencode/releases) 下载二进制文件。

## 配置

配置 API 密钥以使用任意 LLM 提供商。

推荐使用 [OpenCode Zen](/zh/docs/zen)（团队验证的模型列表）。

1. 在 TUI 中运行 `/connect`，选择 `opencode`，访问 [opencode.ai/auth](https://opencode.ai/auth)。
2. 登录并添加计费信息，复制 API 密钥。
3. 粘贴密钥。

或者选择其他提供商，详见 [提供商配置](/zh/docs/providers)。

## 初始化

进入项目目录：

```bash
cd /path/to/project
opencode
```

运行：

```bash
/init
```

这会生成 `AGENTS.md` 文件，帮助 OpenCode 理解项目结构。

## 使用

现在可以自由提问！

### 提问示例

```text
How is authentication handled in @packages/functions/src/api/index.ts
```

### 添加功能

1. **创建计划**：按 **Tab** 切换到计划模式，描述需求。
2. **迭代计划**：提供反馈或图片。
3. **构建**：切换回构建模式，执行更改。

### 进行更改

直接要求更改，例如：

```text
We need to add authentication to the /settings route...
```

### 撤销/重做更改

- `/undo`：撤销
- `/redo`：重做

### 分享对话

`/share` 生成分享链接。
