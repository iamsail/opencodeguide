# 配置 (Config)

## 这是什么？
OpenCode 提供了丰富的配置选项，允许开发者根据个人习惯和项目需求定制工具的行为。配置文件通常以 YAML 或 JSON 格式存在，涵盖了模型选择、界面主题、快捷键绑定以及自定义规则等多个方面。

## 什么时候你会用到它？
- 当你需要切换默认使用的 LLM 模型（例如从 GPT-4 切换到 Claude 3.5）。
- 当你需要设置 API 密钥或代理服务器地址以解决网络连接问题时。
- 当你需要自定义快捷键或调整界面外观以匹配你的开发环境时。

## 核心概念
- **全局配置**：适用于所有项目的通用设置，通常存储在用户主目录下。
- **项目级配置**：仅针对当前项目生效的设置，通常存储在项目根目录的配置文件中。
- **环境变量**：通过系统环境变量传递敏感信息（如 API Key）。
