# Opencode + Claude 使用指南（中文）

本指南面向 **第一次使用 opencode 的中文用户**。

---

## Opencode 如何使用 Claude

Opencode 通过 **Anthropic API** 调用 Claude。

⚠️ 注意：不能使用 Claude Code 专用凭证。

---

## 配置步骤

```bash
export ANTHROPIC_API_KEY=你的_API_Key
```

然后运行：

```bash
opencode
```
